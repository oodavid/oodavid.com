Molecule Match is now Button Up!

I've had a lot of great feedback about Molecule Match, but a recurring gripe was that the learning curve was steep. Which has led to rather a few changes:

  • I've ditched the chemical theme in favour of the haberdashery
    • Atoms are now Buttons
    • Bonds are now Threads
    • Molecules are now Patterns
    • Cherries are now Yarn
  • The Tutorial has been improved (again)
    • I think the Tutorial will need constant updates as it's critical for onboarding players
  • I've cobbled together a quick website
    • buttonupgame.com
    • Finally gave Google AppEngine a whirl, love it.
  • I've enlisted some help! The game no longer looks or sounds like a coder made it
    • Simon James - audio
    • Gavin Patrick McPhail - visual

What do you think? Ready to play?
